# PINGuim

PINGui(m) é um pequeno programa em python que realiza um teste de ping e apresenta os seus resultados gráficamente.

## Requisitos

Este programa foi feito em python3.8.

```bash
pip3 -r requirements.txt
```

## Usar

```python
python3 pinguim.py
```
